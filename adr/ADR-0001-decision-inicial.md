# ADR-0001: Decisión inicial de la práctica

## Contexto
Este repositorio sirve para practicar un workflow integrado (Issue → PR → CI) con evidencia mínima.

## Decisión
Usar:
- C4 (docs/c4) para comunicar arquitectura.
- ADRs (docs/adr) para decisiones.
- OpenAPI (docs/api/openapi.yml) como contrato de API.

## Consecuencias
Los PRs que cambien arquitectura o API deben actualizar estos artefactos.
